# e-plantShopping application
The application will allow user to browse a variety of house plants and add them to a shopping cart. The shopping cart would allow user to view their items and their total cost as well as to adjust the number of items in the cart before checkout. 

There are 3 pages :
Page 1 : Landing page
-   A brief introduction to the company, what they do and their mission.
-   A Get Started button to go to the product listing page

Page 2 : Product Listing
-   User can browse the house plants which are grouped by common feature, such as air purify or aromatic, in their product line. 
-   User can see the unit price of each plant and a button to add it to the shopping cart.
-   In the page header user can navigate to the shopping cart page by clicking on the icon of a shopping cart. Clicking on the company logo will direct back to the landing page
-   The current number of items is displayed in the shopping cart icon

Page 3 : Shopping Cart
-   User can see the plants that they have selected and option to increase or decrease the number of the selected plants as well as to delete it from the shoppimg cart. 
-   User can also see the subtotal cost of each plants and the total cost for all the plants in the cart. 
-   A contine shopping button to go back to produt listing to continue shopping and a Checkout button.# coding-project-template
